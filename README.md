# NBA Alley Oop

## Description

An application for users to learn about NBA teams. The user can choose a specific team and then be taken to a page with information about the team selected.

# Built With

- HTML
- CSS
- Materialize
- Javascript
- JQuery
- NBA Team Info API
- Wikipedia API
